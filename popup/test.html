Actions Test Config Docs About

    The test runs all test urls against all defined actions.
    The display is grouped by url and action group.
    Per action group the actions are listed pointing to the transformed url.
    The last part of the line indicates which transformations were applied.
    eg. [0,1] means the first two transformations were applied.
    [] indicates that no transformation was applied.

    Add some testURLs to the config to see test result.
